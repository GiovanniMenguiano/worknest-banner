# WorkNest — Banner de reserva

Maqueta HTML + CSS del banner de estado de reserva para WorkNest (Util Challenge Product Design).

## Vista previa

Abre [`index.html`](./index.html) en el navegador o usa un servidor local:

```bash
cd worknest-banner
python3 -m http.server 8080
```

Luego visita: http://localhost:8080

## Contenido

- **5 estados**: próxima reserva, check-in, por expirar, finalizada, cancelada
- **Tokens de color** alineados con las variables de Figma (modo claro)
- **Cuenta atrás** animada en estados activo y por expirar
- Demo interactiva: clic en el banner o en los puntos inferiores
